Barre d'en-tête
---------------

.. figure:: ../../Images/headerbar_img1.png
   :alt: Figure 4

À côté du logo TYPO3, vous trouvez les fonctions suivantes :

#. Votre nom d'utilisateur et le bouton de déconnexion
#. L'ajout aux favoris et les espaces de travail
#. La recherche

Pour des raisons de sécurité, la session expire après quelques minutes d'inactivé. L'affiche suivant permet alors de
vous reconnecter :

.. figure:: ../../Images/expire_img1.png
   :alt: Figure 5

Vous pouvez actualiser la session mais les modifications non enregistrées seront perdues. ATTENTION: enregistrez vos
modifications régulièrement. Pour rappel, TYPO3 enregistre les données et les fichiers sur le serveur et non sur votre
ordinateur personnel. Vous pouvez actualiser la session, mais les modifications non enregistrées seront perdues - gardez
donc à l'esprit d'enregistrer vos modifications régulièrement. TYPO3 enregistre les données et les fichiers sur le
serveur et non sur votre PC.
